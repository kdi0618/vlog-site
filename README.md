## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Second, create a file `.env.local` from `.env.local.tmp`.

Third, Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
